Introduction

Bootstrap is an extremely popular front-end framework for developing on the web. Used for marketing pages, web applications, and everything in between, it makes building websites fast and easy. To use Bootstrap all you need to know is the basics of HTML, CSS, and a little patience to read through the documentation.

This tutorial will take you through the basics of using the Bootstrap CSS framework to develop a simple webpage, like a resume. This was written using Bootstrap 4 alpha 6, but the few small diferences from Bootstrap 3 are noted as they come up.

  • A container has many rows
  • A row has many columns
  • A column may have many rows

Working with Columns

Bootstrap uses a 12-column grid system. Column elements may span multiple columns to create a wide variety of layouts.

  • The width of the column is specified by adding a number at the end of the class name. (Example .col-3)
  • Columns can be indented by using the offset class. (Example .offset-4)
Bootstrap 3 Notes
  • Bootstrap 3 requires the use of responsive sizes, which we will cover next (.col-md-3)
  • Bootstrap 3 uses a slightly different class for offsets (.col-md-offset-3)

Responsive Layouts

Bootstrap is made for creating responsive websites. There are five different screen sizes that you can use with your columns to make your layout responsive:

  • xs (no size specified) – 0px - 575px – Extra small screen / phone
  • sm – 576px - 767px – Small screen / phone
  • md – 768px - 991px – Medium screen / tablet
  • lg – 992px - 1199px – Large screen / desktop
  • xl – 1200px+ – Extra large screen / wide desktop
  • Note: Bootstrap 3 uses the -xs- size in its classes and does not have an xl class, see the documentation for details

These classes combine with .col and .offset to apply at certain screen sizes. When the screen is smaller than any columns you have defined, it will take the full width of the screen. What does this mean? Let's look at a few examples of class declarations, and what it will look like at each screen size. Change the size of your browser to see how these examples look at each breakpoint.

Example 1 .col-3

.col-3

This targets screens that fall into the xs range. It will be 3 columns wide at all screen sizes.

Watch as the column changes sizes as the size of your browser changes.

Protip! Building layouts in Bootstrap

When initially working on your layout, make all columns using the .col-lg-# size (.col-md-# in v3). This lets you get a basic layout designed for desktops, but it will collapse to every column taking the full screen width on smaller screens which is often the desired behavior.
Once you have finished your website, you can go back and tweak the column sizes for smaller screens if needed to add that final polish.

Headings

Bootstrap has styles for all headings sizes, h1 through h6. These work without any classes on the heading elements, and can also be applied to p elements using the classes .h1 through .h6. Optionally, you can add smaller, secondary text next to the headings as well.
